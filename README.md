# Master Scraper

Centralized lottery results scraper using Playwright and Supabase.

## Features
- Scalable architecture to support multiple lottery websites.
- Instagram scraping with automated image detection.
- Master Supabase integration for a single source of truth.

## Setup
1. `npm install`
2. Create `.env` from `.env.example`
3. Add your Master Supabase credentials.
4. `npm start`
